Rooted Rebellion

This project is a 2D fighting game created with Unity. It was made in 48 hours for the 2023 Winter Global Game Jam. This was a two-person project so source control via GitHub was used for development. My contributions involved the art, animations, and the boss AI's functionality.

The goal for players is to defeat the tree boss. They have 2 attacks: axe throwing and meleeing. The boss can shoot branches at the player and has an area-of-effect attack. As the boss loses health they become more aggressive.
