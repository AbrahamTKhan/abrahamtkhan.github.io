Merch Store

This project is a website with store functionality, where users can browse the available merch and add things to their basket. PHP was used to allow for the implementation of dynamic features, this includes retrieving and displaying information from databases.

The page's elements were added with HTML and formatted with CSS. SQL is used to fetch store items where the user can filter the results to determine how they're displayed. The website functions in various dimensions/devices.

There are various pages including home, store, login, basket, and more.
